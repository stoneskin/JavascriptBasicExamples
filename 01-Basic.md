#Javascrpt in Html#
```
<!DOCTYPE html>
<html>
  <head>
    <script>
    console.log('Hello!');
    </script>
  </head>
  <body>
    <h1>Our Page</h1>
    <script>
      console.log('In Body');
    </script>
    <script type="text/javascript"src="js/01-basic-1.js">
      console.log('Hello in Body!');
    </script>
  </body>
</html>
```
